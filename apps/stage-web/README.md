## Feature Flags

| Flag | Default | Description |
|------|---------|-------------|
| `VITE_FEATURE_SOCIAL` | `false` | Enable social module (Twitter integration). When enabled in development, the social nav entry appears in Settings → Modules. In production builds, this is disabled by default. |
